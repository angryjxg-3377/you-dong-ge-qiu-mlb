## MLB Betting Rules

### The MLB Betting Calendar
```
Spring Training : February → March (low weight — starters limited)
Regular Season : Late March → September (162 games per team)
Wild Card Games: October (single elimination)
Division Series : October (Best of 5)
Championship    : October (Best of 7, ALCS/NLCS)
World Series    : October (Best of 7)

Key dates:
  Trade deadline : Late July → bullpen upgrades shift totals
  September rosters: September 1 → expanded rosters (25→28)
```

### Bet Type 1: Moneyline
```
Most popular MLB bet — pick the outright winner

Format (American odds):
  Favourite : -160 (bet $160 to win $100)
  Underdog  : +140 (bet $100 to win $140)

MLB moneyline is unique:
  Underdogs win ~44% of games — highest in North American sports
  → Never blindly bet heavy favourites (-200 or worse)
  → Underdog value hunting is the core MLB strategy

Key rule: Moneyline bet is ACTION if game starts
  If listed starter doesn't pitch → bet may be cancelled
  (depends on sportsbook "listed pitcher" vs "action" rules)
```

### Bet Type 2: Run Line (ATS)
```
Baseball's version of the point spread
Almost always set at ±1.5 runs (unlike NFL/NBA variable spreads)

Example:
  Dodgers -1.5 (-130) vs Yankees +1.5 (+110)
  → Dodgers must win by 2+ runs to cover -1.5
  → Yankees cover if they lose by 1 run or win outright

Alternative run lines (some sportsbooks):
  -2.5 / +2.5 — for very heavy favourites

Run line juice flips:
  Moneyline favourite -180 → Run line -1.5 may be +110
  (Getting +1.5 on the underdog often better value than ML)

⚡ Key MLB insight:
  Run line underdogs (+1.5) cover more often than
  moneyline underdogs win outright
  → +1.5 underdog is one of the most popular MLB bets
```

### Bet Type 3: Over/Under (Run Total)
```
Combined runs scored by both teams (9 innings)

MLB totals typically: 7.0 to 10.5 runs
League average ~8.8 runs per game (2025-26)

Example:
  Total: 8.5
  Over: Combined 9+ runs
  Under: Combined 8 or fewer

Key total factors (MLB-specific):
  Starting pitcher ERA / WHIP / recent form
  Bullpen ERA (second-most important)
  Ballpark factor (Coors Field vs Petco Park)
  Wind direction and speed (CRUCIAL)
  Temperature (cold = less offense)
  Umpire total tendencies (some call strikes wider = Under lean)
```

### Bet Type 4: First 5 Innings (F5)
```
⚡ MLB-unique bet type — most popular with sharp bettors ⚡

Bet on only the first 5 innings — removes bullpen variance

F5 bet types:
  F5 Moneyline : Which team leads after 5 innings
  F5 Run Line  : -0.5 / +0.5 spread (who's ahead after 5)
  F5 Total     : Over/Under for just first 5 innings

Why F5 is popular:
  Starting pitcher is the biggest single factor in MLB
  Eliminates unpredictable bullpen meltdowns
  More predictable outcome based on SP matchup

F5 Total typically: 3.5 to 5.5 runs

Key F5 factors:
  SP ERA vs league average (first 5 innings ERA)
  SP pitch count limit (if SP maxes out at 4 innings → irrelevant)
  Opposing lineup vs SP handedness (L/R splits)
```

### Bet Type 5: Player Props
```
Individual player performance bets

Pitcher props (most popular):
  Strikeouts (K props):
    SP strikeouts Over/Under (e.g. Scherzer Over 7.5 Ks)
    Most important: SP K rate vs opposing lineup K rate
  Earned runs allowed
  Outs recorded / innings pitched
  Hits allowed

Batter props:
  Home run (anytime HR):
    e.g. Judge to hit HR at +320
    Most popular individual player prop in MLB
  Total bases (TB):
    e.g. Ohtani Over 1.5 total bases
  Hits (e.g. Over 0.5 hits = gets at least 1 hit)
  RBIs / Runs scored
  Stolen bases

NRFI / YRFI (fastest growing MLB prop):
  NRFI = No Run First Inning
  YRFI = Yes Run First Inning
  Bet on whether EITHER team scores in the 1st inning
  
  NRFI factors:
    Both SP's first inning ERA
    Lead-off hitter speed/OBP
    Ballpark (some parks more first-inning prone)
```

### Bet Type 6: Parlay
```
Combine 2+ picks — all must win

MLB parlay typical odds:
  2-team : ~+260
  3-team : ~+600
  4-team : ~+1200

Same-Game Parlay (SGP):
  Combine outcomes from ONE game
  e.g. Dodgers ML + Over 8.5 + Freeman Over 1.5 TB
  Correlated SGP: Dodgers win big → Freeman likely had good game

Round Robin:
  3-8 teams, create every combination of 2-3 team parlays
  Partial hit still pays out on winning combinations
```

### Bet Type 7: Team Totals
```
Bet on just ONE team's runs scored (not combined)

Example:
  Dodgers Team Total: Over/Under 4.5 runs
  → Just the Dodgers — doesn't matter what Yankees score

When to use:
  Strong pitcher on one side → bet Under on OPPOSING team's total
  Weak bullpen on one side → bet Over on that team's total
  Platoon advantage one-sided → Over/Under on favoured team

More precise than game total in asymmetric matchups
```

### Bet Type 8: Series Betting
```
Bet on who wins a 2-3 game series (not individual game)

Most popular: Friday night opening game series
  → Who wins the 3-game weekend series?

Series total:
  How many games will the series go? (2 or 3 games)

Advantages:
  Smooths out single-game variance
  SP rotation matters across series
  Home/away within series manageable
```

### Bet Type 9: Futures
```
Season-long bets

Most popular:
  World Series winner
  AL / NL pennant winner
  Division winner
  MVP (Ohtani, Judge, Acuna etc)
  Cy Young Award
  Team win total (Over/Under X wins)
  Rookie of the Year

Best value: Opening day or after April slump/surge
```

### Bet Type 10: Live / In-Game Betting
```
Real-time betting during the game

Most popular live markets:
  Next inning total (over/under)
  Next team to score
  Adjusted run line and total
  Will this at-bat result in a hit/HR/strikeout?

Key MLB live signals:
  SP struggling early + weak bullpen → live Over
  Strong SP dominant + good bullpen → live Under
  SP pulled early → re-evaluate F5 live line
  Injury to key batter mid-game → Under lean
```

## 15-Factor Four-Layer Analysis Model

### Core Layer (Contribute to Score — High Weight)

#### Factor 1: Starting Pitcher Analysis
```
⚡ Most important single factor in MLB betting ⚡
Starting pitcher accounts for ~40% of game outcome

Data: BALLDONTLIE player stats

Analyze both SPs:

Recent form (last 5 starts):
  ERA (earned run average)
  WHIP (walks + hits per inning)
  K/9 (strikeouts per 9 innings)
  BB/9 (walks per 9)
  HR/9 (home runs allowed per 9)
  Average innings per start (does SP go deep or short?)

Situational stats:
  Home vs away ERA splits
  vs Left-handed batters / vs Right-handed batters
  vs Opposing lineup this season (if previous meetings)
  Recent pitch count (fatigued from last start?)
  Days rest (4 days = normal, 5+ = extra rest = good)

Starting pitcher rating:
  Ace (ERA < 2.80, WHIP < 1.05) + facing weak lineup → ⭐⭐⭐ Under
  Solid SP (ERA 3.20-3.80)                           → ⭐⭐
  SP struggling (ERA > 4.50 recently)                → ❌ Lean Over
  Spot starter / bullpen game                        → 🔴 High variance

⚠️ Lineup handedness vs SP:
  Left-handed SP vs right-heavy lineup → SP advantage
  Right-handed SP vs left-heavy lineup → SP advantage
  Platoon advantage = additional -0.5 runs on total
```

#### Factor 2: Bullpen Analysis
```
Data: BALLDONTLIE team stats + recent game logs

Second most important MLB factor

Analyze both bullpens:
  Bullpen ERA (season + last 7 days)
  Bullpen WHIP
  Recent usage (days pitched in last 3 days)
  Closer availability (has team used closer 2+ days straight?)
  High-leverage reliever status

Bullpen fatigue calculation:
  Sum of relievers' appearances in last 3 days
  0-1 appearances total → ✅ Fresh bullpen
  2-4 appearances       → 🟡 Moderate use
  5+ appearances        → 🔴 Overworked → lean Over

Key insight:
  SP goes deep (7+ innings) → bullpen factor low
  SP leaves early (4-5 innings) → bullpen factor critical

Bullpen game (no announced SP):
  → Automatic High Variance flag
  → Total leans slightly Over (multiple arms = less consistent)
```

#### Factor 3: Ballpark Factor
```
Data: Historical run totals per ballpark (built into model)

MLB ballparks vary dramatically in run production
This is unique to baseball — no equivalent in NFL/NBA

High offense parks (Over lean +0.5 to +1.5 runs):
  Coors Field (Colorado Rockies) — altitude, thin air → biggest Over park
  Great American Ball Park (Cincinnati)
  Fenway Park (Boston) — small dimensions, Green Monster

Low offense parks (Under lean -0.5 to -1.0 runs):
  Petco Park (San Diego) — marine layer, large dimensions
  Oracle Park (San Francisco) — marine layer, cold
  T-Mobile Park (Seattle) — marine layer, pitcher friendly
  Dodger Stadium (LA) — pitcher friendly, especially for LHP

Neutral parks:
  Most other MLB stadiums → minimal adjustment

Park factor adjustment:
  Coors Field → add 1.5 runs to total baseline
  Petco/Oracle → subtract 0.7 runs from total baseline
```

#### Factor 4: Lineup Analysis
```
Data: BALLDONTLIE roster + injury report + beat reporters

Key lineup factors:
  Star batter availability:
    MVP-caliber batter OUT → subtract 0.8-1.2 runs from team total
    Multiple key bats OUT → significant total impact

  Lineup construction:
    Right-handed heavy lineup vs LHP → advantage to lineup
    Left-handed heavy lineup vs RHP → advantage to lineup
    Balanced lineup → no significant split advantage

  Recent offensive form (last 7 days):
    Team batting average
    OPS (on-base + slugging)
    Runs per game
    Strikeout rate (high K team vs high K pitcher = Under lean)

  Late lineup scratches:
    Day-of lineup changes → significant line movement trigger
    Check official lineup ~3-4 hours before first pitch
```

#### Factor 5: Head-to-Head History
```
Data: BALLDONTLIE historical games

MLB H2H analysis:
  Last 5 meetings (same ballpark preferred)
  Scoring tendencies in the series
  SP vs lineup historical performance
  Division rivals → play 6-7 times/year → rich H2H data

Key H2H signals:
  Team consistently scores against this SP → Over lean
  SP dominates this lineup historically → Under lean
  One team consistently covers run line vs this opponent
```

#### Factor 6: Advanced Metrics
```
Data: BALLDONTLIE advanced stats

MLB-specific advanced metrics:

  wOBA (Weighted On-Base Average):
    Better measure of offensive quality than batting average
    .320+ = above average offense
    .350+ = elite offense

  xFIP (Expected Fielding Independent Pitching):
    Normalises HR allowed for luck
    Better predictor of future SP performance than ERA

  BABIP (Batting Average on Balls in Play):
    .300 = league average
    Team BABIP much higher/lower → regression expected

  Strikeout rate vs Walk rate (K/BB):
    SP K/BB > 3.0 = dominant control pitcher

  Hard Hit Rate:
    % of batted balls hit hard against SP
    High hard-hit rate + low ERA → ERA will rise (regression)
```

### Adjustment Layer (Modify Weights — Medium Impact)

#### Factor 7: Weather Analysis
```
Data: OpenWeatherMap /forecast (match city + first pitch time)

⚡ MORE CRITICAL IN MLB THAN ANY SPORT ⚡

Wind direction (most important):
  Wind blowing OUT to center field (e.g. 15+ mph):
    → Fly balls carry further → HRs more likely → Over lean
    → Add 0.5-1.0 run to total
    → Coors Field + out wind = massive Over lean

  Wind blowing IN from center (15+ mph):
    → Fly balls die → HRs suppressed → Under lean
    → Subtract 0.5-0.8 run from total

  Wind > 20 mph any direction:
    → High scoring variance regardless of direction
    → Wide range of outcomes

  Crosswind: minimal scoring impact

Temperature:
  Below 50°F (10°C): cold air = less carry on fly balls
    → Subtract 0.3-0.5 runs from total
  Below 40°F: significant cold impact
    → Subtract 0.7-1.0 runs, pitcher-friendly conditions
  Above 90°F: hot, humid conditions
    → Ball carries better, slight Over lean

Rain:
  Light drizzle: minimal impact
  Heavy rain risk: game may be delayed/postponed
    → Monitor radar, wait for official weather report

⚡ Wind Rule Summary:
  Out 15+ mph  → +0.8 runs to total
  Out 20+ mph  → +1.2 runs to total
  In 15+ mph   → -0.7 runs from total
  Cold < 50°F  → -0.4 runs from total
  Cold < 40°F  → -0.8 runs from total
```

#### Factor 8: Umpire Analysis
```
Data: TheRundown API / umpire assignment data

Home plate umpire is announced day of game

Key umpire stats (season averages):
  Runs per game in his games (vs league avg ~8.8)
  Strikeout rate in his games
  Called strike zone size (wide zone = more Ks = Under)
  BB rate in his games

Umpire tendencies:
  Known tight zone (umpire calls lots of walks):
    → More baserunners → more scoring → Over lean
    → Add 0.3-0.5 runs to total

  Known wide zone (umpire punishes hitters):
    → More strikeouts → less contact → Under lean
    → Subtract 0.3-0.5 runs from total

  High total umpire (his games average 9.5+ runs/game):
    → Strong Over lean
  Low total umpire (his games average 8.0- runs/game):
    → Strong Under lean

Note: Umpire impact is real but not dominant
Combine with other factors — don't bet solely on umpire
```

#### Factor 9: Home/Away Factor
```
Data: BALLDONTLIE team stats (home/away splits)

MLB home advantage: ~0.5-1 run less than NFL/NBA
Baseball home advantage is smaller but real

Team home vs away SP ERA:
  Some SPs are dramatically better at home
  → Check SP home ERA vs away ERA specifically

Travel factors:
  West coast to East coast (3-hour time zone change):
    → Morning games for West coast teams (body clock)
    → Slight fatigue, especially for position players

  Long road trip (7+ games away):
    → Performance tends to decline by games 5-7

Home team lineup advantage:
  Batting last = strategic advantage in close games
  Walk-off wins → home team wins tied games more often
  Factor into run line analysis
```

#### Factor 10: Recent Form & Momentum
```
Data: BALLDONTLIE game logs (last 10 games)

Team offensive form (last 10 games):
  Runs scored per game
  OPS trend (improving/declining)
  Runs with RISP (runners in scoring position)
  Recent HR rate

Pitching form (last 10 games):
  Team ERA allowed
  Bullpen ERA last 7 days specifically

Hot/cold streaks:
  Team on 7+ game win streak:
    → Regression likely in next 2-3 games
    → Fade heavy favourite on win streak

  Team on 5+ game losing streak:
    → Due for positive regression
    → Value on underdog

Momentum signals:
  Team just shut out (0 runs) → typically bounces back offensively
  Team just scored 10+ runs → regression toward mean
```

### Background Layer (Display Only — No Calculation Impact)

#### Factor 11: Sharp Money & Line Movement
```
Data: The Odds API (line movement) + TheRundown

MLB sharp money signals:

SP-driven line movement:
  SP confirmed → line moves immediately
  SP scratched → line pulled or moves dramatically
  This is expected movement → not "sharp money"

True sharp money signals:
  Line moves WITHOUT SP news → sharp action
  Run line moves independent of moneyline → sharp run line play
  Total moves without weather change → sharp total play

Reverse line movement:
  65%+ public on Dodgers but line moves TOWARD Yankees
  → Classic sharp fade → lean Yankees

Steam moves:
  Rapid simultaneous movement across books
  → Large syndicate bet → follow direction

Signal tiers:
  Reverse line movement + low public % → ⭐⭐⭐ Sharp signal
  Line movement after SP confirmation  → 🟡 Expected, not sharp
  No movement                          → Neutral
```

#### Factor 12: Injury Report
```
Data: BALLDONTLIE injuries + beat reporter updates

MLB injury designations:
  10-Day IL (Injured List) → confirmed missing 10+ days
  15-Day IL → more serious
  Day-to-Day → could play tonight or be scratched

Position player impact:
  Superstar (30+ HR pace) OUT → -0.8 runs team total
  Star (20+ HR pace) OUT      → -0.5 runs team total
  Role player OUT              → minimal impact

⚠️ Late scratches (day of game):
  Most impactful — line won't fully adjust yet
  Check beat reporters on Twitter/X for lineup updates
  ~3-4 hours before first pitch = official lineup posted

SP on IL:
  → Game pulled or replacement SP → re-analyze
```

#### Factor 13: Platoon & Matchup Data
```
Data: BALLDONTLIE player stats (career splits)

Platoon advantage analysis:
  Left-handed SP vs opposing lineup:
    Count left-handed hitters in lineup
    LHB vs LHP historically worse → disadvantage to lineup

  Right-handed SP vs opposing lineup:
    Count right-handed hitters in lineup
    RHB vs RHP historically worse → disadvantage to lineup

  Switch hitter consideration:
    Switch hitters neutralize platoon disadvantage

Specific batter vs specific SP (if faced before):
  Career stats of lineup's key hitters vs tonight's SP
  Particularly useful for props (K props, total bases)

Note: Display as context — influences confidence in totals/props
```

#### Factor 14: Travel & Schedule Strength
```
Data: BALLDONTLIE schedule

MLB schedule grind (162 games):
  Day games after night games ("day-after"):
    → Fatigue factor, especially for position players
    → Slight Under lean for offensive output

  Series opener vs series finale:
    Series opener: SP is set, best available
    Series finale: SP may be weaker (rotation order)

  Back-to-back series in different cities:
    Multiple time zone changes in one week
    → Cumulative fatigue by series 3-4

  September expanded rosters:
    September 1 → 28-man rosters (up from 26)
    Fresh relievers available → Under lean (more quality arms)
```

#### Factor 15: Motivation & Standings
```
Data: BALLDONTLIE standings

MLB motivation factors:

Playoff race:
  Wild Card race (within 3 games of spot) → HIGH motivation
  Division lead chase → HIGH motivation
  Every win matters in September

Eliminated teams:
  Mathematically eliminated → motivation drops
  Young players get opportunities → higher variance
  → Lean Over (bad pitching) but less predictable

Interleague considerations:
  AL team at NL park (or vice versa) → DH rule differences
  AL teams without DH at NL park historically struggle slightly

Season series finale:
  Last meeting of season in division → higher intensity
  Rivalry games (Yankees/Red Sox etc) → slightly elevated scoring

Note: Motivation is a smaller factor in MLB (162 games)
than NFL (17 games) — display as context
```

## Total (Over/Under) Specific Model

```
MLB totals are the most analytically driven bet in all sports

Step 1: Calculate baseline total
  (SP1 ERA normalized) + (SP2 ERA normalized) + park factor

Step 2: Weather adjustment
  Apply wind direction/speed and temperature adjustments

Step 3: Umpire adjustment
  Apply umpire run tendency vs league average

Step 4: Bullpen adjustment
  Fresh vs overworked bullpens affect late-inning scoring

Step 5: Compare to set total
  Model projection vs bookmaker line → find edge

F5 total model:
  Focus only on both SPs (ignore bullpen)
  F5 total = approximately 55-60% of full game total
  Strong SP matchup → F5 Under lean even if full game is neutral
```

## NRFI/YRFI Special Model

```
NRFI = No Run First Inning (currently most popular MLB prop)
YRFI = Yes Run First Inning

Factors for NRFI:
  Both SP first-inning ERA (most important)
  Lead-off hitter OBP and power (top of lineup)
  Ballpark (some parks more first-inning prone)
  Umpire zone (wide zone → more Ks → NRFI lean)

NRFI probability estimate:
  Both SPs have low first-inning ERA (<1.50) → NRFI ~65%
  One SP has high first-inning ERA (>3.00) → YRFI lean
  League average: ~53% of innings have no scoring

Output format:
  NRFI: [Probability]% — [Recommendation]
  YRFI: [Probability]% — [Recommendation]
```

## Three Betting Strategies

### Safe Strategy
```
Selection criteria:
  Composite score ≥ 75
  Both SPs confirmed and healthy
  No bullpen game flag
  No extreme weather (wind < 15mph)
  Preferred: F5 moneyline or run total Under with ace pitchers

Goal: Highest hit rate — exploit SP matchup edges
```

### Balanced Strategy
```
Main picks: Composite score ≥ 65
Value picks (1-2 legs):
  Run line underdog (+1.5) with value
  NRFI in dominant SP matchup
  Team total Under for weak offense vs ace

Target total odds: Safe odds × 1.5~2.5x
```

### Value Strategy
```
Main picks: Composite score ≥ 55
Contrarian picks:
  Underdog moneyline with SP edge
  Over in Coors Field with out wind
  Reverse line movement play
  SP strikeout prop with strong K matchup
  NRFI/YRFI prop for additional legs

Target total odds: Safe odds × 3x+
```

### Execution Flow
```
Step 1: Read bets.json → find status=pending

Step 2: BALLDONTLIE /mlb/games?id=ID
  Only process if game_status = Final

Step 3: Reconcile each bet
  Moneyline: Winner takes it
  Run line: Apply ±1.5 to final score
    Dodgers -1.5: Dodgers must win by 2+
  Total: Sum both teams' runs
    8.5 total: 4+5=9 → Over wins

Step 4: Update bets.json + stats.json

Step 5: Output result card with special situation tracking
  (Did wind call work? Did NRFI hit?)
```

## API Call Optimization

```
Per game: ~5-7 calls
  1. BALLDONTLIE /mlb/games      → Schedule + scores
  2. BALLDONTLIE /mlb/stats      → SP stats (both pitchers)
  3. BALLDONTLIE /mlb/injuries   → Injury report
  4. The Odds API /odds          → Lines from 15+ books
  5. OpenWeatherMap /forecast    → Wind + temperature (CRITICAL)

Optional:
  6. TheRundown /umpires         → Umpire assignment + stats
  7. BALLDONTLIE /mlb/box_scores → Advanced metrics

BALLDONTLIE free tier → sufficient for daily analysis
The Odds API 500/month → ~70 games per month (15/day × 3-4 days)
OpenWeatherMap 1000/day → ample for all games
```
